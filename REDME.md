# Marketing Funnel & Conversion Performance Analysis Dashboard

## Project Overview

This project analyzes marketing campaign performance using the Bank Marketing dataset. An interactive Power BI dashboard was developed to evaluate customer conversion rates, marketing funnel performance, and campaign effectiveness.

## Objectives

* Analyze marketing funnel performance
* Calculate customer conversion rates
* Identify customer drop-off points
* Compare campaign performance across different customer segments
* Provide actionable business recommendations

## Dataset

**Dataset:** Bank Marketing Dataset (bank-additional-full.csv)

The dataset includes customer demographics, marketing campaign information, contact methods, previous campaign outcomes, and subscription status.

## Tools Used

* Microsoft Power BI
* Microsoft Excel
* Bank Marketing Dataset

## Dashboard Features

* KPI Cards
* Funnel Analysis
* Conversion Rate Analysis
* Contact Method Analysis
* Job-wise Analysis
* Monthly Campaign Performance
* Education Analysis
* Previous Outcome Analysis
* Interactive Slicers

## Key Insights

* Measured overall customer conversion rate.
* Compared campaign performance across customer segments.
* Identified factors influencing customer subscriptions.
* Evaluated previous campaign effectiveness.

## Recommendations

* Target high-performing customer groups.
* Improve campaign timing.
* Optimize communication methods.
* Personalize marketing strategies.
